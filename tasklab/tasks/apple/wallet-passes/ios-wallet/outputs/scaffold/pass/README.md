# Pass bundle

This folder holds the source files that get packaged into a `.pkpass`.

Typical required files include:

- `pass.json` (generated by `11-render-pass-json.sh`)
- `icon.png` (generated by `10-generate-placeholder-images.sh`, replace with real art)
- `logo.png` (optional but common)

Build scripts also create:

- `manifest.json`
- `signature` (signed build only)
